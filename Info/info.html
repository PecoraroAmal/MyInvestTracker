Portfolio Tracker

Gestisci i tuoi investimenti in modo privato e sicuro

Privacy Garantita

Questa webapp ti permette di tenere traccia dei tuoi investimenti in modo completamente privato. Tutti i dati rimangono sul tuo dispositivo, nessuna informazione viene salvata sui nostri server.

Editor

Modifica graficamente il tuo portfolio. Al primo caricamento sembrerà vuoto, ma appena aggiungerai un nuovo asset il tuo JSON prenderà forma.

Tracker

Visualizza graficamente il tuo portfolio. Carica il JSON modificato nell'editor per vedere tutte le statistiche e i grafici dei tuoi investimenti. Include dati di default che vengono sovrascritti caricando il tuo JSON senza interferenze.

PAC

Simula un Piano di Accumulo Capitale. Nota: potrebbero esserci errori nei calcoli, ci scusiamo per eventuali imprecisioni.

Variazione

Calcolo veloce della variazione percentuale di un asset. Inserisci solo due valori (prima e dopo) per ottenere il risultato.

Calcolatrice

Effettua la ripartizione ottimale del tuo portfolio in base alle percentuali desiderate e al budget disponibile. Include dati di default che vengono sovrascritti caricando il tuo JSON senza interferenze.

Come Iniziare

1

Scarica il Template JSON

Scarica il file template per iniziare a costruire il tuo portfolio

2
